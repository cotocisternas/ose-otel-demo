## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTTP request → baggage context | X-Customer-Tier header value enters the application and is set as baggage without validation |
| baggage context → AMQP message headers | W3CBaggagePropagator serializes the baggage entry into the `baggage:` W3C header injected into RabbitMQ message properties |
| AMQP message headers → consumer span attributes | TracingMessageListenerAdvice extracts baggage; BaggageSpanAttributeProcessor stamps it as span attributes in Tempo |
