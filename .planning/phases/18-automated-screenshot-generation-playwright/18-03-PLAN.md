**If the script exits non-zero**, examine the summary table output. Common failure modes and fixes:
- "Grafana not reachable" → run `mise run infra:up` first
- "waitForSelector timeout" on step-11-* → check `docker logs ose-otel-collector` for errors; tail_sampling needs traces in the buffer; re-run with `WARMUP_MS=60000 mise run screenshots`
- "otel-collector health check timed out" → wait 30s and re-run; the Collector may have a slow startup after restart
- "node: command not found" → `mise exec -- node --version` to verify Node is on PATH
